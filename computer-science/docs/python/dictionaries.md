### Initialization

- Initialize a [dictionary](computer-science/docs/basics/data-structures/dictionaries.md) like this:
```python

my_dict = {
    "name": "Bob",
    "age": 30,
    "city": "los Angeles",
}

```

- Or using the `dict` constructor
```python
another_dict = dict(name="Bob", age=30, city="Los Angeles")
```
