## Compare two values
with multiple conditions

```python
if x > y:
	# do something
elif x < y:
	# do something else
else:
	# previous conditions not met
```

## Chain conditions

### 1. and

```python
if x > y and x == y:
	# both conditions are met
```

### 2. or

```python
if x > y or x == y:
	# one or more of the conditions are met
```
