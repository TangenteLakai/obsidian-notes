
- Basic comment:
```c
// This is a comment
```

- Inline comment:
```c

```

- Comments to split you code into individual segments for example:
```c
/*************************************************************
 * Prototypes
 ************************************************************/
```

- Defining function parameters/return value and their use-case:
```c
/** 
 * Brief description.
 *
 * @param a first parameter
 * @return if any return value
 * 
 * Detailed description
 **/
```
