### 7. **UPPER**

- **Purpose**: Converts a string to uppercase.
- **Example**:
		
		```sql

		SELECT UPPER(name) AS uppercase_name

		FROM employees;

		```c
		

- **Output**: Returns all names in uppercase.

These keywords are widely used in SQL for data analysis and transformation tasks.
