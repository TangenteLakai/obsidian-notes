## Compare two values

```c
if (x > y) {
	// do something
}
else if (x < y) {
	// do something else
}
else {
	// previous conditions not met
}
```

## Chain conditions

### 1. and

```c
if (x > y) && (x == y) {
	// both conditions are met
}
```

### 2. or

```c
if (x > y) || (x == y) {
	// one or more of the conditions are met
}
```
